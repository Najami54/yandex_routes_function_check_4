# yandex_routes_function_check_4

## Описание
Репозиторий содержит артефакты QA по проекту "Яндекс.Маршруты" (версия 2.0):
- Тест-план
- Тест-кейсы
- Баг-репорты
- Скриншоты
- Mindmap

## Окружение тестирования
- ОС: Windows 10 22H2, build 19045.6093  
- Браузеры: Windows Firefox 33, Яндекс.Браузер  
- Разрешения экрана: 800x600, 1920x1080

## Структура репозитория
- `test-plan.md` — тест-план
- `mindmaps/` — mindmap по заданиям
- `test-cases/` — тест-кейсы для заданий 2,4,5
- `bug-reports/` — баг-репорты и скриншоты
- `postman/` — Postman коллекции
- `miscellaneous/` — прочие документы
