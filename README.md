## Структура проекта

# Mindmaps
*Расположены в папке mindmaps/.
*Визуализируют структуру проекта, связи между компонентами и основные идеи.

# Тесты и тест-кейсы
*Основные файлы находятся в папке test-cases/:
*test-cases.xlsx — основной набор тест-кейсов.
*testing.xlsx — дополнительные тестовые сценарии.
*Содержат описания тестов, шаги воспроизведения и ожидаемые результаты.

# Баг-репорты

*Расположены в папке bug-reports/.
Файлы:
bug-reports.xlsx — таблица зарегистрированных багов.
testit_import_test_yaroute.xlsm — импорт багов в систему тестирования.

#Скриншоты и анализ
В папке bug-reports/screenshots/.
Используются для анализа ошибок и проверки функциональности через DevTools и Charles.

#Прочие документы
miscellaneous/report.pdf — общий отчёт по проекту.

#Папка test-documentation/ содержит:
*README.md — данный файл документации.
*test-plan.md — план тестирования, описание подходов и критериев.

## Проверка функциональности Яндекс.Маршруты
Окружение:
ОС: Windows 10 22H2, сборка 19045.6093
Браузеры: Яндекс.Браузер (разрешение 800x600), Firefox (разрешение 1920x1080)

### Основные действия и результаты:
*Актуализирована документация по экрану заказа такси, обновлены сведения о добавлении и подтверждении номера телефона.
*Проверены формы добавления прав и карт в каршеринге:
*Создана таблица принятия решений.
*Проведены тесты в двух окружениях.
*Выявлены ошибки в функциональности кнопок.
*Тест валидации формы “Добавление карт”:
*Обнаружены ошибки валидации полей “Код” и “Номер карты”.

-- Фронтенд практически не реагирует на некорректные символы; рекомендуется пересмотреть логику валидации на фронте и бэке.

### Тест новой функциональности аэротакси при нерабочем бэкэнде:
*Добавлена новая иконка для типа транспорта.
*Настроен расчет стоимости и времени поездки.
*Проведена автоматизация и проверка по чек-листу.

### Итоги:
Все тесты выполнены, выявлено значительное количество багов.
Сервис выполняет базовый набор функций, но не готов к полноценному релизу.

Выявленные ошибки могут негативно повлиять на пользовательский опыт и репутацию компании.
